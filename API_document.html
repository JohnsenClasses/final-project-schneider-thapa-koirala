Get Weather Info API Documentation

How does our API work?

Currently, Get Weather Info API provides hypothetical sensor data (temperature and humidity) at two locations: UGA, Athens (Lat: 33.948 Lon: -83.377) and UGA, Tifton (Lat: 31.479 Lon: -83.521). In the near future, more locations will be added.

Our API is formatted in a structure known as JSON.

Get Weather Info Endpoints

  • Use this host_url to access the main webpage

    • https://getweatherinfo.ga

  • API's interactive user interface (FastAPI's docs)

    • https://getweatherinfo.ga/wAPI/docs

  • Access all sensor data for both locations

    • Humidity and temperature data of every two-minutes interval for both locations.

    Example of the request: https://getweatherinfo.ga/wAPI/sensor_data

    Example of the response:

    [ { "ID": 249, "Latitude": 33.948, "Longitude": -83.377, "Temperature": 82, "Humidity": 43, "Timestamp": "2022-05-04 18:06:05" }, { "ID": 250, "Latitude": 31.479, "Longitude": -83.521, "Temperature": 89, "Humidity": 58, "Timestamp": "2022-05-04 18:06:07" }, { "ID": 251, "Latitude": 33.948, "Longitude": -83.377, "Temperature": 80, "Humidity": 43, "Timestamp": "2022-05-04 18:06:10" }
  • Access humidity and temperature data for a particular location
    • Currently, API's has sensor data from two locations; UGA, Athens (Lat: 33.948 Lon: -83.377) and UGA, Tifton (Lat: 31.479 Lon: -83.521).

    https://getweatherinfo.ga/wAPI/sensor_data/{Latitude}/{Longitude}

  • Access humidity and temperature data for provided date to current date

    User input datetime format; 2022-05-04 18:11:52

    • https://getweatherinfo.ga/wAPI/sensor_data/{Date}

  • Access humidity and temperature data between two time points

    User input datetime format; 2022-05-04 18:11:52

    • https://getweatherinfo.ga/wAPI/sensor_data/between/{FromDate}/{ToDate}

  • Access humidity and temperature data for a particular day

    User input datetime format; 2022-05-04

    • https://getweatherinfo.ga/wAPI/sensor_data/of_a_day/24hours/{Date}

  • Access minimum temperature and humidity for a particular day

    User input datetime format; 2022-05-04

    • https://getweatherinfo.ga/wAPI/sensor_data/of_a_day/24hours/{Date}/temperatureHumidity/Min"

  • Access maximum temperature and humidity for a particular day

    User input datetime format; 2022-05-04

    • https://getweatherinfo.ga/wAPI/sensor_data/of_a_day/24hours/{Date}/temperatureHumidity/Max"

  • Access average temperature and humidity for a day

    User input datetime format; 2022-05-04

    • https://getweatherinfo.ga/wAPI/sensor_data/of_a_day/24hours/{Date}/temperatureHumidity/Avg"


Description of the variables used for the API

Variable Unit Description
Lat, Lon degree latitude and longitude of the location
Temperature farhenheit temperature of the location
Relative humidity percentage relative humidity of location
Date year:month:day h:m:s specific date and time
