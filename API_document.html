Get Weather Info API

Currently, Get Weather Info API provides hypothetical sensor data (temperature and humidity) at two research locations: UGA, Athens (Lat: 33.948 Lon: -83.377) and UGA, Tifton (Lat: 31.479 Lon: -83.521). In the near future, more research locations will be added.

Get Weather Info Endpoints

  • API documentation

    • https://getweatherinfo.ga/wAPI

  • API's interactive user interface (FastAPI's docs)

    • https://getweatherinfo.ga/wAPI/docs

  • Access all sensor data from all locations

    • https://getweatherinfo.ga/wAPI/sensor_data

  • Access sensor data for a particular location
    • Currently, API's has sensor data from two locations

    https://getweatherinfo.ga/wAPI/{Latitude}/{Longitude}
